# Job Search Analytics Tracker

## Overview
This project is a structured system I built to track and analyze my job search performance while pursuing a Bachelor’s degree in Cybersecurity and industry certifications.

The goal is to treat the job search process like a data problem by tracking measurable outcomes and identifying patterns that improve resume effectiveness and interview response rates.

## Problem
Most job seekers track applications, but not outcomes. This makes it difficult to understand what is actually working.

## Solution
I created a tracking system that records key metrics across every application to identify patterns and improve results over time.

## Data Points Tracked
- Resume version used for each application
- Total number of applications submitted
- Number of interview responses received
- Number of interviews scheduled
- Number of job offers received
- Job description patterns that correlate with interview responses

## Goals
- Improve resume targeting based on real response data
- Identify which job descriptions and keywords perform best
- Increase interview response rate through iterative optimization
- Build a data driven approach to career development
